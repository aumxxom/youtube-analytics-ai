# 📊 YouTube Analytics AI

This project analyzes YouTube channel performance using Python and machine learning.

## 🚀 Features

* Fetch video data
* Engagement analysis
* Risk prediction
* Trend visualization

## 📦 Requirements

* Python
* Libraries in requirements.txt

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Open the notebook:
   youtube_analytics_ai.ipynb

3. Run all cells

## 🔑 API Key

Set your YouTube API key as environment variable.
